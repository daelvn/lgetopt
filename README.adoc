lua-getopt
==========

Command-line argument parser for Lua 5.3.


Features
--------

 - GNU-style flag handling (`-rcsuv` == `-r -c -s -u -v`) (only for booleans)
 - link:[D-style] option definitions (more or less)


Usage
-----

It's just one function:

[source,lua]
----
local getopt = require('getopt')

getopt(arg, ...)
----
